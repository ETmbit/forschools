# ETmbit lesmodules

![ElecTricks](/content/modules.png)

In de rechter bovenhoek bevindt zich de modulenaam en het volgnummer van de opdracht. Deze komen overeen met die op de routekaart.

## Document-versies
De opdrachten zijn op A5-formaat vormgegeven. Als boekje nemen ze weinig ruimte in. Er staat wellicht ook al een robot naast de computer.

De bladzijden van de *...-B.pdf*-versies zijn als boekje gerangschikt. Gebruik deze versies om leerlingboekjes te printen.
**Printereigenschappen**: dubbelzijdig afdrukken via de korte zijde.<br>
Nieten kan met een langarm-nietmachine, bijvoorbeeld https://www.manutan.nl/nl/mnl/langarm-nietmachine-rapid-hd12.

De versies eindigend op *...-L.pdf* bevatten losbladig gerangschikte pagina’s. Deze versies zijn vooral geschikt voor presentaties of voor weergave in github.
