## Document-versies
De opdrachten zijn op A5-formaat vormgegeven. Als boekje nemen ze weinig ruimte in. Er staat wellicht ook al een robot naast de computer.

In de versies *et-nl-module-...-L.pdf* zijn de bladzijden losbladig gerangschikt. Deze versies zijn vooral geschikt om voor uitleg met bijvoorbeeld met een beamer te vertonen.

In de versies *et-nl-module-...-B.pdf* zijn de bladzijden als boekje gerangschikt. Gebruik deze versies om leerlingboekjes te printen.
**Printereigenschappen**: dubbelzijdig afdrukken via de korte zijde.
Nieten met een lange nietmachine, bijvoorbeeld https://www.manutan.nl/nl/mnl/langarm-nietmachine-rapid-hd12.
