# ETmbit Buggy



## Veilig in gebruik

![Buggy Init](/content/buggy-field.png)

Module **BGGY-1** laat leerlingen kennismaken met het veilig gebruik van de buggy. De bijbehorende MakeCode-uitbreiding is zo ontworpen dat de buggy niet wil rijden zolang de veldkleur niet is ingesteld. Zodra de veldkleur is gekozen, stopt de buggy automatisch wanneer hij een andere kleur detecteert, tenzij voor die kleur expliciet een actie is geprogrammeerd.

![Buggy Init](/content/buggy-color.png)

De lijn neemt daarbij een bijzondere plaats in. Ook voor de lijn moet een kleur worden ingesteld. De buggy kan vervolgens in twee verschillende modi werken:

![Buggy Init](/content/buggy-border.png)

1. **De lijn als begrenzing van het veld**<br>
De lijn vormt de rand van het werkgebied. Zodra de buggy buiten het veld komt, stopt deze automatisch.

2. **De lijn volgen**<br>
De lijn wordt gebruikt als pad om te volgen. In deze modus blijft de buggy rijden zolang hij de lijn detecteert en stopt hij juist wanneer hij van de lijn af raakt.
