# Het ETmbit-project [GB](README.EN.md)
Het ETmbit-project biedt een complete leerlijn robotica en maakt het lesgeven eenvoudig en toegankelijk. Het project is gericht op:
- bovenbouw basisonderwijs 
- onderbouw voortgezet onderwijs 
Er wordt gewerkt met **micro:bit + MakeCode**. Robotonderdelen worden **3D-geprint** of schaf je **kant-en-klaar aan**. Van leerkrachten en docenten wordt niet meer dan een **minimale voorkennis** verwacht.

---
## :twisted_rightwards_arrows: Lesplan
ETmbit heet gedifferentieerd leren als uitgangspunt. Leerlingen kiezen hun eigen route door het lesmateriaal op basis van interesse, niveau en leerstijl. Leerkrachten en docenten gebruiken een volgsysteem om de ontwikkeling te begeleiden.
